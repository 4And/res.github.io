Настройка репозиториев для обновлений

Редактируем sources list

nano /etc/apt/sources.list

Добавляем эти данные ниже содержимого файла:

# Proxmox VE pve-no-subscription repository provided by proxmox.com
# NOT recommended for production use
deb http://download.proxmox.com/debian/pve bookworm pve-no-subscription

# Security updates
deb http://security.debian.org/debian-security bookworm-security main contrib

Отключаем репозитории для Production
Следующими отключаем платные репозитории в pve-enterprise.list:

nano /etc/apt/sources.list.d/pve-enterprise.list

Закомментировать эту строку:

#deb https://enterprise.proxmox.com/debian/pve bookworm pve-enterprise
Важно! После изменения репозиториев выполните: 
apt update && apt upgrade -y
